# corogue [![1]][2]

An experimental Game Boy Advance game written in Rust.

> The crate that facilitates running Rust code on the GBA is vendored from
Theo Belaire's [rusty-TONC] project.


## Requirements

- [xargo]
- devkitARM
- libgba
